# 2025年度 卒業論文：[北白川C式・中津Ⅰ式と初期称名寺式の系統関係　—属性数量分析による近畿・瀬戸内・南西関東の比較研究—]

## 概要 (Overview)
このリポジトリは、[早稲田大学文学部考古学コース] の卒業論文「[北白川C式・中津Ⅰ式と初期称名寺式の系統関係　—属性数量分析による近畿・瀬戸内・南西関東の比較研究—]」で使用した実験コードおよび分析スクリプトである。

[本研究では、中期末～後期初頭にかけての初期称名寺式期・中津式成立期の土器片を分析し、対象遺跡の基礎的統計量の要約やGower距離とMCAを用いた高次元の類似度計算とt-SNEによる二次元上での土器片クラスター分布の図化を行った]

## 環境等 (Environment)

本研究におけるデータ分析および実装は、Windows 11 上で構築された Python 3.13.5 環境にて実施した。

データの整形および数値計算には、Pandas (ver 2.3.2) および NumPy (ver 2.3.3) を使用した。 統計解析および機械学習の実装には、scikit-learn (ver 1.7.2) を中心に用いた。具体的には、欠損値の補完に IterativeImputer、次元圧縮に t-SNE、クラスタリングに K-means法を適用した。

また、混合データの非類似度計算（Gower距離）には Gower (ver 0.1.2)、多変量解析には Prince (ver 0.16.1)、統計的検定には SciPy (ver 1.16.2) を使用した。

分析結果の可視化には、Matplotlib (ver 3.10.6)、Seaborn (ver 0.13.2)、および Plotly (ver 6.3.0) を適宜使い分けた。なお、グラフ内の日本語フォント対応として japanize-matplotlib (ver 1.1.3) を、画像出力の補助として Kaleido (ver 1.2.0) を併用した。

## ファイル構成 (File Structure)
- `cleaning1.py`: データのクリーニングやダミー化を行います。数字通りに実行してください。
- `cleaning2.py`: データのクリーニングやダミー化を行います。数字通りに実行してください。
- `cleaning3.py`: データのクリーニングやダミー化を行います。数字通りに実行してください。クリーニング終了後、データが二重になっている場合は同一部分を消去してください。

- `clossanalysis.py`: クロス集計表を作成します。
- `make_pie_chart.py`: 積み上げグラフを作成します。

- `requirements.txt`: 依存ライブラリ一覧
- `motodata.csv`: 欠測値・誤り値修正済みの生データ
